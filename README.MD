# GreenMind 🌱

## 📖 :brazil: Sobre o Projeto | ## :us: 📖 About The Project

🇧🇷
GreenMind é um aplicativo móvel desenvolvido como parte do Projeto Integrador da UNIVESP. O objetivo é criar uma ferramenta que utiliza Inteligência Artificial para ajudar horticultores, agricultores familiares e entusiastas da jardinagem a identificar espécies de plantas, diagnosticar problemas de saúde vegetal e receber sugestões de cuidados.

Este protótipo inicial estabelece o fluxo de usuário principal, desde a tela inicial até a apresentação de um resultado de análise simulado, focando na experiência do usuário e na viabilidade da aplicação.

:uk: :us:
GreenMind is a mobile application developed as part of the UNIVESP Integrator Project. The goal is to create a tool that uses Artificial Intelligence to help horticulturists, family farmers, and gardening enthusiasts to identify plant species, diagnose plant health issues, and receive care suggestions.

This initial prototype establishes the main user flow, from the home screen to the presentation of a simulated analysis result, focusing on the user experience and application feasibility.

## 🚀 Como Executar o Protótipo | ## 🚀 Getting Started with the Prototype

:brazil:
1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/nelsonsouzajr/GreenMind
    ```
2.  **Navegue até a pasta do frontend:**
    ```bash
    cd GreenMind-Projeto/frontend
    ```
3.  **Instale as dependências:**
    ```bash
    npm install
    ```
4.  **Inicie o projeto:**
    ```bash
    npx expo start -c
    ```
